============
Introduction
============

Target group: **Administrators**

What does it do?
================

This extension provides several services to cleanup your TYPO3 instance.

It implements the following possibilities to execute cleanup methods:

* backend module
* toolbar item
* processDatamap_afterDatabaseOperations hook
* drawItem hook

Cleanup services are registered in typoscript.
Therefor it is possible to extend the extension with your own services easily.
